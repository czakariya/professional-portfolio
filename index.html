portfolio

Rogue Pickings Website
HTML & CSS from Photoshop Design Comp

This was one of the first websites that I coded while taking an HTML and CSS course with Skillcrush. Although this was a one-page website, it could easily be transformed to a multi-page site. Rogue Pickings is an ideal website for a small local restaraunt that needs to have an online presence. It can easily be customized with different images, fonts, and colors to the branding of the restaraunt.
